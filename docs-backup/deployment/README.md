# 部署文檔

本目錄包含系統部署相關的所有文檔。

## 📄 文檔列表

### 主要部署文檔
- **DOCKER_DEPLOYMENT.md** - Docker 部署詳細說明（最重要）
- **QUICK_START.md** - 快速開始指南
- **DEPLOYMENT_README.md** - 部署總結

### 部署狀態
- **DEPLOYMENT_SUMMARY.md** - 部署狀態總結
- **DEPLOYMENT_FINAL_STATUS.md** - 最終部署狀態
- **DEPLOYMENT_SUCCESS.md** - 部署成功說明
- **DEPLOYMENT_STATUS.md** - 部署狀態

### 資料庫
- **MIGRATION_INSTRUCTIONS.md** - 資料庫遷移說明

## 📖 使用建議

### 首次部署
1. 閱讀 **QUICK_START.md** 快速開始
2. 參考 **DOCKER_DEPLOYMENT.md** 了解詳細流程
3. 使用 **MIGRATION_INSTRUCTIONS.md** 執行資料庫遷移

### 日常部署
1. 查看 **DEPLOYMENT_SUMMARY.md** 了解當前狀態
2. 參考 **DOCKER_DEPLOYMENT.md** 執行部署

### 故障排除
1. 查看 **DEPLOYMENT_STATUS.md** 了解問題
2. 參考 **DOCKER_DEPLOYMENT.md** 的故障排除章節

## 🎯 推薦閱讀順序

1. QUICK_START.md（快速開始）
2. DOCKER_DEPLOYMENT.md（詳細說明）
3. MIGRATION_INSTRUCTIONS.md（資料庫遷移）
