## 🎯 主要功能模組

系統提供以下核心功能模組：

### 1. KPI 管理
- KPI 字典與定義
- 多種閾值模式（固定值、相對值、預測值）
- 燈號自動計算
- 版本控管

### 2. 戰略管理
- 平衡計分卡 (BSC)
- 戰略儀表板
- 趨勢分析
- 下鑽路徑

### 3. OKR 管理
- 季度目標設定
- 關鍵結果追蹤
- 自動進度計算
- 與 KPI 關聯

### 4. 任務管理 (Kanban)
- 看板視圖
- 拖放操作
- 多維度分組
- 戰略對齊

### 5. PDCA 循環
- 結構化改善流程
- 自動排程回報
- 資料品質檢核
- 改善行動追蹤

### 6. 緊急事件管理
- 事件記錄與追蹤
- 處置清單
- 個資遮罩
- RACI 協作

### 7. RACI 工作流
- 權責分配矩陣
- 可執行工作流
- SLA 檢查
- 稽核軌跡
