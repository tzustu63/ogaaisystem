# 文檔目錄

本目錄包含專案的所有文檔，已按類別整理。

## 📁 文檔分類

### 1. 專案核心文檔 (`/docs/core/`)
- 專案概述、需求文件、開發規範

### 2. 開發文檔 (`/docs/development/`)
- 功能實作狀態、缺失功能報告、測試指南

### 3. 部署文檔 (`/docs/deployment/`)
- Docker 部署、遷移說明、快速開始

### 4. OpenSpec 規格 (`/openspec/`)
- 規格驅動開發相關文檔
